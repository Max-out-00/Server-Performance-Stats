#Server Performance Statistics Script

This Bash script prints a summary of current server performance metrics including CPU usage, disk usage, and memory usage.
It provides a quick, human-readable snapshot for monitoring purposes.

Features

# CPU Usage
Displays user, system, and idle percentages using top

# Disk Usage
Shows total disk space of the root filesystem

Shows available disk space from a mounted drivers directory (if present)

# Memory Usage
Shows total, used, and free memory in human readable format using free -h
